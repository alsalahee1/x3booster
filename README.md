# X3 S-Performance

**Fuel & Engine Oil Booster — a high-conversion product website.**

Marketing and storefront site for **X3 S-Performance**, a two-product line of automotive
chemical additives: a **Fuel Booster** and an **Engine Oil Booster**. The site is built to
turn a skeptical, mobile-first viewer into a WhatsApp inquiry or purchase in a single scroll
session — leaning on cinematic, high-contrast product photography, before/after proof imagery,
and a frictionless cart. Aimed at car owners, workshop operators, and fleet managers.

## ✨ Highlights

- **Two dedicated product pages** — Fuel Booster and Engine Oil Booster, each with its own hero, proof, and specs.
- **Shop + cart** — a `Shop` page with a slide-out `CartDrawer` and cart state managed via React Context.
- **WhatsApp-first conversion** — a persistent floating WhatsApp button for direct inquiries.
- **About & Contact pages** — brand story and contact/lead capture.
- **Motion & polish** — page transitions, scroll-reveal animations, a scroll-progress bar, count-up stat counters, and toast notifications.
- **Mobile-first, accessibility-minded** — designed for WCAG AA contrast on dark backgrounds and `prefers-reduced-motion`, with an Arabic-capable body font for future bilingual copy.

## 🛠 Built With

- **React 18** + **React Router 6** (client-side routing, SPA)
- **Vite 5** (dev server & build)
- **Vanilla CSS** (per-component `.css` files, no CSS framework)
- Google Fonts — *Barlow Condensed* & *Readex Pro*

## 🚀 Running Locally

Requires [Node.js](https://nodejs.org/) (v18+ recommended).

```bash
# Install dependencies
npm install

# Start the dev server (http://localhost:5173)
npm run dev

# Build for production (outputs to dist/)
npm run build

# Preview the production build
npm run preview
```
